# RESOURCES.md 格式

`RESOURCES.md` 是可信资料清单。教学中的事实、定义、机制、最佳实践，应尽量来自这里，而不是凭记忆。

## 模板

```md
# {主题} 资料

## 核心资料

- [{资料名}](https://example.com)
  用途：{什么时候查它}
  可信原因：{官方文档、论文、作者权威、项目源码等}

## 补充资料

- [{资料名}](https://example.com)
  用途：{适合补充哪个角度}

## 社区/实践场

- [{社区名}](https://example.com)
  用途：{适合提问、看真实案例、获得反馈}

## 缺口

- {当前还缺什么资料，后续需要补}
```

## 规则

- 优先官方文档、论文、源码、权威书籍、高质量社区。
- 每个链接都要写“用来解决什么问题”。
- 资料宁少勿滥。5 个高质量来源比 30 个泛泛链接更有用。
- 用户不想看英文资料时，可以用中文解释，但专业名词仍保留常用英文。
- 如果某个结论来自推断，要明确说“这是根据资料推断”。
