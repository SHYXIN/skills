# Learning Record 格式

学习记录放在 `./learning-records/`，文件名按顺序递增：

```text
0001-slug.md
0002-slug.md
```

它不是课堂日志，只记录会影响后续教学的事实：用户已经理解了什么、纠正了什么误区、目标有没有变化。

## 模板

```md
# {短标题}

## 记录
{1-3 句话：用户理解了什么，为什么这会影响后续教学。}

## 证据
{用户如何表现出理解：复述、例子、练习、纠错。}

## 后续影响
{以后可以少讲什么、应该多练什么、下一步适合学什么。}
```

## 什么时候写

- 用户能用自己的话讲清一个关键概念。
- 用户举出了正确例子或完成了小练习。
- 用户原来的误解被修正。
- 用户说明自己已经掌握某个前置知识。
- 学习目标、范围或偏好发生变化。

## 不要记录

- “今天讲了 X”这种流水账。
- 用户只是看过，但没有证明理解的内容。
- 纯旁路问答。旁路问答保存到 `./side-questions/*.html`。
