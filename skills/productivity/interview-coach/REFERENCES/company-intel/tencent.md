# 腾讯 面试情报

## 面试风格

- **基础扎实**：OS、网络、数据库基础必问
- **C++ 岗位多**：如果是 C++ 岗，虚函数/内存管理/STL 必问
- **项目深挖**：会基于简历问很细的项目问题
- **算法中等**：LeetCode Medium 为主，不追求 Hard
- **团队匹配**：三面通常是部门总监，考察团队契合度

## 各轮次重点

| 轮次 | 重点 | 说明 |
|------|------|------|
| 一面 | 基础 + 编码 | 八股文 + 1-2 道算法 |
| 二面 | 深度 + 项目 | 项目深挖 + 系统设计 |
| 三面 | 综合 + 潜力 | 部门总监，考察综合能力 |
| HR 面 | 动机 + 薪资 | 为什么来腾讯、职业规划 |

## 高频考点

### 基础八股（必问）
- TCP/UDP 区别、三次握手四次挥手
- 进程 vs 线程 vs 协程
- 内存管理（虚拟内存、页表）
- MySQL 索引 + 事务 + 锁
- Redis 数据结构 + 持久化

### C++ 专项（C++ 岗）
- 虚函数表原理
- 智能指针（shared_ptr/unique_ptr/weak_ptr）
- STL 容器底层实现
- 内存对齐
- 右值引用和移动语义

### 算法
- 链表操作
- 二叉树
- 动态规划（中等）
- 排序算法

### 系统设计
- IM 系统
- 游戏服务器架构
- 分布式存储

## 文化关键词

- **正直**
- **进取**
- **合作**
- **创新**

## 面试官喜欢的回答特征

1. **基础扎实** — 八股文答得清楚
2. **项目有深度** — 能讲清楚技术选型的原因
3. **有思考** — 不只讲做了什么，还讲为什么这么做
4. **态度谦虚** — 不会就说不会

## 面试官不喜欢的回答特征

1. 基础概念答不上来
2. 项目经历说不清楚
3. 过度包装经历
4. 对技术没有热情
